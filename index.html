CaptainHook

Avast, ye landlubbers CaptainHook is a bloody flexible and easy to use git hook manager.

The Cap'n in a Nutshell

If ye want to make sure nobody commits code that breaks yer tests or codin` standard, or ye want to validate commit messages, CaptainHook can help ye to get going mate. ARRRR!

Available versions

There are two versions of CaptainHook available. One Golang version, that could be installed natively on most machines and a PHP version targeted to PHP developers.
